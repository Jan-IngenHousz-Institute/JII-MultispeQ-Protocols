JII - MultispeQ Protocols
=========================

.. figure:: _static/images/jan-ingenhousz-institute-logo-header.avif

   This is a Project supported and funded by the `Jan IngenHousz Institute <jan-ingenhousz-institute.org>`_.

.. rubric:: About

This is a collection of prototcols that can be used together with the `JII-MultispeQ package <https://github.com/jan-ingenhousz-institute/JII-MultispeQ>`_ 
to take measurements using the MultispeQ.

Along with protocols for the MultispeQ, the package also provides a validation tools, so the protocol code can be tested
for protential issues before running it on a MultispeQ.

.. note::
   Contributions of new protocols are welcome! They can be submitted via a pull-request.
